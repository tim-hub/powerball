# Behavior Rules

Supplements the Claude Code default system prompt with what's distinctive about how I want you to work.

1. **Think before coding** — surface tradeoffs, don't hide confusion · @references/think-before-coding.md
2. **Simplicity first** — minimum code, no speculative abstractions · @references/simplicity-first.md
3. **Surgical changes** — every changed line traces to the request · @references/surgical-changes.md
4. **Goal-driven execution** — define success, loop until verified · @references/goal-driven-execution.md
5. **Agent consultation** — escalate to Opus or `powerball-personal:senior` when uncertain · @references/agent-consultation.md
6. **Testing** — prefer single tests over full suite · @references/testing.md
